Introduction

OIIA Network is an Ethereum Proof-of-Stake (PoS) Layer 1 blockchain featuring the meme token OIIA OIIA (Spinning Cat).

Oiia Network uses Ethereum clients Geth + Lighthouse as initial nodes, without modifying the core code.

Network Specifications

Initial Validators

Oiia Network launched with 128 validators, stored in the genesis.ssz file.

Initial Token Supply

Oiia Network does not pre-allocate tokens to developers or DAOs.

The initial supply is 4096 OIIA, all post-launch circulation comes from mining rewards.
